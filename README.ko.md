## 1. 목적

본 개인정보 처리방침은 당사가 제공하는 Android 애플리케이션 "아바타" (이하 "본 앱") 사용 시 정보를 수집, 이용, 공유 및 보호하는 방식을 설명합니다. 본 앱을 사용함으로써 본 방침에 따른 정보 처리에 동의하시는 것으로 간주됩니다.

본 앱은 **B2B 기업용 애플리케이션**으로, 조직 관리자의 등록 (디바이스 코드 + 비밀번호) 후 시설 단말기로 운영됩니다.

## 2. 수집하는 정보

### 2.1 자동 수집

| 종류 | 수집 시점 | 목적 |
|------|----------|------|
| **음성 데이터** | "대화 시작" 버튼 누른 후 마이크로 취득 | 실시간 AI 대화, 음성 인식 |
| **카메라 영상** (선택) | 사용자가 카메라를 명시적으로 활성화한 경우만 | 시각적 보조 |
| **디바이스 식별자** | 앱 최초 실행 시 | 디바이스 인증, 조직 연동 |
| **앱 사용 로그** | 모든 기능 사용 시 | 장애 대응, 서비스 개선 |
| **네트워크 정보** | WebSocket 연결 시 | 통신 품질 모니터링 |
| **대략적인 위치 정보** | 디바이스 부팅 시 (선택) | 디바이스 위치 관리 |

### 2.2 사용자 제공 정보

- **디바이스 코드 + 비밀번호**: 조직 관리자가 발급
- **설정**: 언어, 음성, 지식 베이스 할당 등

### 2.3 수집하지 **않는** 정보

- 방문자 이름, 주소, 전화번호, 이메일
- 신용카드 및 금융 정보
- 생체 정보 (얼굴 인식 데이터, 지문 등)
- 연락처, SMS, 통화 기록

## 3. 정보의 이용 목적

수집한 정보는 다음 목적으로만 이용합니다:

- 본 앱의 핵심 기능 제공 (AI 대화, 디지털 휴먼 영상 전송)
- 장애 원인 분석 및 복구
- 서비스 품질 개선 및 신기능 개발
- 부정 이용 감지 및 방지
- 법령 준수 (세무, 소비자 보호 등)

다음 목적으로는 **이용하지 않습니다**:
- 타겟 광고
- 제3자 판매
- 사용자 프로파일링

## 6. 정보 보안

- **전송 암호화**: 모든 통신은 TLS 1.2 이상으로 암호화
- **저장 암호화**: 서버 측 데이터는 AES-256으로 암호화
- **접근 제어**: 인증된 당사 직원만 접근 가능
- **사고 대응**: 중대 유출 발생 시 72시간 이내 고객 및 당국에 통지

## 7. 사용자의 권리

귀하 (또는 귀하 조직의 관리자) 는 다음 권리를 보유합니다:

- **접근권**: 당사가 보유한 사용자 정보의 공개 청구
- **정정권**: 부정확한 정보의 정정 청구
- **삭제권** ("잊혀질 권리"): 정보의 삭제 청구
- **데이터 이동권**: 기계가 읽을 수 있는 형식으로 정보 수령 청구
- **이의 제기권**: 정보 처리에 대한 이의 제기

청구 방법: **privacy@paripi-erp.com**
원칙적으로 30일 이내에 회신합니다.

## 8. 카메라·마이크 (Android 권한)

본 앱은 다음 Android 권한을 사용합니다:

- **마이크 (RECORD_AUDIO)**: AI 대화에 **필수**. 앱 시작 시 사용자 허가 요청
- **카메라 (CAMERA)**: 선택. 사용자가 화면의 카메라 버튼을 누르고 명시적으로 동의한 경우만 작동. **기본값 꺼짐**
- **위치 (ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION)**: 조직 관리자가 디바이스 위치 관리에 사용
- **블루투스 (BLUETOOTH_*)**: 관련 기기 (섀시 등) 와의 연동

## 9. 어린이 개인정보
